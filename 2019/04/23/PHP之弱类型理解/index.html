在PHP中有两种比较的符号 == 与 ===，一种是松散的，一种是严格的。
== 在进行比较的时候，会先将字符串类型转化成相同，然后再比较。
=== 在进行比较的时候，会先判断两种字符串的类型是否相等，然后再比较。

官方文档的链接为：http://php.net/manual/zh/types.comparisons.php
